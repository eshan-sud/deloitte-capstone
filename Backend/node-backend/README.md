# Node Backend - Notification Service Scaffold

Basic Express service for notification workflows in the Event Management System.

## Included Middleware

- `cors`
- `helmet`
- `morgan`
- `compression`
- `cookie-parser`
- `express-rate-limit`
- `dotenv`

## Run

```bash
cd Backend/node-backend
npm install
npm run dev
```

## Endpoints

- `GET /api/health`
- `POST /api/v1/notifications/test`
